UPnP GPS Device

Description
This bundle provides a UPnPDevice which provides GPS positions from a GPS receiver connected to a serial port.
For more details on how to use this bundle, refer to the /doc/readme.html file embedded in the bundle jarfile.

start http://www-adele.imag.fr/users/Didier.Donsez/dev/osgi/domowareutil/domowareutil.jar
start http://www-adele.imag.fr/users/Didier.Donsez/dev/osgi/upnpgenutil/upnpgenutil.jar

rem start http://www-adele.imag.fr/users/Didier.Donsez/dev/osgi/upnpposition/upnpposition.jar

obr start "Tiny Shell GUI"
